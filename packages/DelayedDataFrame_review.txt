## R/

* Perhaps you can use something like a `cbind.DelayedDataFrame` in the
constructor function (L:68) so that you don't have to ensure that your output
will be `DelayedDataFrame` (L:74)
* `check.names` works when _not_ doing the `cbind` in the contructor. To make it
work with `DelayedDataFrame` inputs, use `make.names` after `cbind`ing.
* The `setAs,DataFrame,DelayedDataFrame` function can be reduced to one line by
using `length(from)`:
```
ldf <- length(from)
lazyIndex <- .LazyIndex(vector("list", as.numeric(as.logical(ldf))),
    index=rep(1L, ldf))
```
* You're using the copy-replace method in DelayedDataFrame-method.R-L:64. It
might not make much of a difference in newer version of R but you should use
the allocate-fill method.
* Can you use `replaceSlots` in `.cbind_DDF` rather than initialize? I think
this is preferred. Thoughts @hpages?
