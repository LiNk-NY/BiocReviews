These can be arguments to a function if the name is not specified.
* Check for `is.missing` rather than `is.null` when no default value is used
in `check_gaussians`.
* Vectorize or loop where possible to avoid long `if` `else` statements
(e.g., in `check_gaussians` you can use an lapply or vapply to check if any of
those fields are missing).
* Perhaps you should keep `clean_profiles` and related operations endomorphic,
meaning they accept and return `MSnSet`.
* Consider grouping functions into one file rather than having 1 file for each
function (for example, AIC related functions; data docs).
* Avoid using vague argument names such as 'input'
* Use switch to avoid a series of `if` `else` statements in
`predict_interactions`

```
interactFUN <- switch(classfier,
    NB = predict_NB_ensemble,
    RF = predict_RF_ensemble,
```
