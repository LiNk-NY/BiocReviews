to read and review.

show the functionality of your package. `eval=FALSE` chunks can become stale
quite quickly.

* You can use the data package for your examples, if necessary. Avoid using
the `tempdir()` especially for downloaded data as pointed out above.
* Some of the logic can be improved, for example:
```r
  MethSM_minus = MethSM[apply(MethSM[,IsMinusStrand], 1, function(i){sum(is.na(i)) != length(i)}) > 0, IsMinusStrand]
  MethSM_plus = MethSM[apply(MethSM[,!IsMinusStrand], 1, function(i){sum(is.na(i)) != length(i)}) > 0, !IsMinusStrand]
```
